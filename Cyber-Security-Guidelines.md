
# Cyber Security Guidelines – Reflection

## Common Cyber Security Threats

In a remote work environment, common threats include:
- phishing emails and fake login pages
- weak or reused passwords
- unsecured Wi-Fi networks
- malware from suspicious downloads or links
- unauthorized access due to unlocked devices

## Best Practices for Security

To keep devices and accounts secure:
- use strong, unique passwords for each account
- enable two-factor authentication (2FA)
- keep software and systems updated
- avoid clicking on suspicious links or attachments
- use secure networks and avoid public Wi-Fi when possible

## Importance of Locking Devices

Locking your computer when away prevents unauthorized access to sensitive data. Even short periods away from your desk can expose your system if left unlocked.

## Handling Phishing and Suspicious Links

If I encounter suspicious emails or links:
- I will not click on them
- verify the sender before taking action
- report the issue if necessary
- delete or ignore the message

## Strong Passwords and Password Managers

A strong password should:
- be long and complex
- include a mix of letters, numbers, and symbols
- not be reused across accounts

Using a password manager helps:
- securely store passwords
- generate strong passwords
- reduce the risk of reuse

## Importance of Two-Factor Authentication (2FA)

2FA adds an extra layer of security by requiring a second verification step (e.g., phone code or authenticator app). It should be enabled for all important accounts.
